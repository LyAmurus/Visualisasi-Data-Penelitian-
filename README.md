## 📌 Deskripsi

**Visualisasi Data Penelitian** adalah proyek skripsi yang bertujuan untuk menyajikan data penelitian dalam bentuk visual yang menarik, interaktif, dan informatif. Proyek ini dibuat untuk membantu pembaca memahami data dengan lebih mudah melalui grafik, chart, dan elemen visual lainnya.

---

## 🛠️ Teknologi yang Digunakan

- 📊 **Chart.js / D3.js** *(atau sesuai yang kamu pakai)*
- 💻 **HTML, CSS, JavaScript**
- 📁 **Node.js / Python** *(opsional, jika ada backend)*

---

## ✨ Fitur Unggulan

- Interaktif visualisasi data
- Tampilan modern dan responsif
- Pengolahan data otomatis
- Export hasil visualisasi

---

## 🎓 Tujuan Skripsi

> "**Membuat media bantu visualisasi data untuk mempermudah pemahaman hasil penelitian.**"
